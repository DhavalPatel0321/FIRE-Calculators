# Fat FIRE

Fat FIRE is Traditional FIRE with a higher annual spending assumption. In v1, the default Fat expense level is **$150,000 per year**, and the same safe withdrawal rate input is used across all variants.

The v1 formula is:

`fatFireNumber = fatExpenses / safeWithdrawalRate`

At the 4% default safe withdrawal rate, the $150,000 default creates a $3,750,000 Fat FIRE target.

## When it fits

Fat FIRE is useful when the goal is work-optional wealth without giving up a high-spending lifestyle. It can represent expensive metro areas, private school, high travel budgets, second homes, or a larger margin for discretionary spending.

## What to watch

Some planners prefer lower withdrawal rates for high-spending or very long retirements. v1 keeps the same safe withdrawal rate across variants for simplicity, so users who want a more conservative Fat FIRE target should lower the SWR slider manually.
